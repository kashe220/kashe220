SDLC: Systems Development Life Cycle

A conceptual model used in project managemnt that describes the stages involved in an information system development project, from an initial feasibility study through maintenance of the completed application.

Various SDLC methodologies

  • Waterfall Model
  • Rapid Application Development RAD
  • Joint Application Development JAD
  • The Fountain Model
  • The Sprial Model
  • Build and Fix
  • Synchroize-and-Stabilize
